# 📘 AI-Powered PDF to Excel Data Extraction System

This project automatically converts **unstructured PDF documents** into a fully structured **Excel file** using a Large Language Model (LLM).  
The system intelligently extracts **key:value pairs**, adds **contextual comments**, and preserves the **exact original wording** from the PDF while organizing everything into a clean table.

## 🚀 Features

- ✔ Converts unstructured PDF text into a structured Excel format  
- ✔ Automatically generates keys — **no hardcoded schema**  
- ✔ Extracts **100%** of the content from the input PDF  
- ✔ Keeps original sentences (no paraphrasing)  
- ✔ Adds a **Comments** column for contextual understanding  
- ✔ Handles multi-line text values cleanly  
- ✔ Simple, readable, and modular code  
- ✔ Works on any similar text-based PDF
